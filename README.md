# sales-data-eda-python
Exploratory Data Analysis of Sales Data using Python
🎯 Objective

Analyze sales data to uncover trends, customer behavior, and revenue drivers

📊 Dataset
Source:  simulated 
Size: 3000+ rows

🛠 Tools Used
Python
Pandas
Matplotlib / Seaborn

Key Insights
Top 20% of customers generate 65% of revenue
Sales peak in Q4 (seasonality)
Certain products consistently underperform
